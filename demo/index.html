This is a demo of the accessible dialog.

Launch Dialog (basic) - This will launch a basic dialog. Since it is non-modal, none of the accessibility enhancements apply
Launch Dialog (animated)- This will launch a basic dialog that is animated. Since it is non-modal, none of the accessibility enhancements apply
Launch Dialog (Form container) - this will launch a dialog with a form in it. It is modal and receives all of our accessibility stuff
Launch Dialog (message) - this will launch a text dialog. It is modal and receives all of our accessibility stuff. It also has a custom role.
